# Proyecto_API

Aplicación Flutter que consume una API de Rotten Tomatoes para mostrar información de películas.

## Tecnologías
- Flutter
- Dart
- API Rotten Tomatoes
- HTTP Requests
- JSON

## Descripción

Esta aplicación permite consultar información de películas usando una API externa.  
Se mostrarán datos como:

- Título
- Año
- Género
- Actores
- Director
- Poster
